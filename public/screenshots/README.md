# Screenshots Directory

This directory should contain three screenshot images:

1. **popup.png** - Screenshot of the extension popup showing:
   - The main toggle switch
   - Statistics display
   - Options link

2. **spell-checking.png** - Screenshot showing:
   - A text editor or form field
   - Red underlines on misspelled words
   - The extension working in action

3. **context-menu.png** - Screenshot showing:
   - Right-click context menu
   - Spelling suggestions
   - "Add to Dictionary" and "Ignore" options

## Recommended Specifications

- **Format**: PNG with transparency or JPG
- **Aspect Ratio**: 16:10 (e.g., 1280x800, 1920x1200)
- **File Size**: Optimize for web (under 500KB each)
- **Style**: Should match the dark theme of the website

## How to Add Screenshots

1. Take screenshots of the extension in action
2. Crop and resize to 16:10 aspect ratio
3. Optimize images (use tools like TinyPNG or ImageOptim)
4. Save as:
   - `popup.png`
   - `spell-checking.png`
   - `context-menu.png`
5. Place all three files in this directory
6. The website will automatically display them

## Temporary Placeholders

If screenshots are not available, the website will show styled placeholder boxes with icons and text.
